あきる野青年会議所 2026 年度「専務理事対応（AI活用）」プロジェクト

更新日: 2026-05-05
対象ツール: Claude Code・Codex・Cowork（全ツール統一）

========================================
## 命名パターン

[進捗アイコン][種別アイコン] [20字以内の本質] — [詳細補足]

========================================
## 3 つのカテゴリ分類

【種別 1】専務のプロジェクト
- 優先度: 最高（🔴）
- 進捗管理: Notion
- 進捗アイコン: ✅完了 / 🚀進行中 / ⏳保留中
- 性質アイコン: タスク内容に応じて（📋📊📝等）
- 例: ✅📋 LOM発送準備 — 資料・スケジュール確認
- 例: 🚀📊 月次レポート — 進捗・予算・人事

【種別 2】専務のタスク（プロジェクト外含む）
- 優先度: 高（🟡）
- 進捗管理: Notion
- 進捗アイコン: ✅完了 / 🚀進行中 / ⏳保留中
- 性質アイコン: タスク内容に応じて（📋📊📝等）
- 例: ✅📝 委員会資料 — 議題・提案内容
- 例: ⏳📋 週次レポート — LOM進捗・委員会連絡

【種別 3】環境開発・改善系
- 優先度: ベストエフォート（🟢）
- 進捗管理: GitHub Issues・Chat
- 期限: なし
- 進捗アイコン: ✅完了 / 🚀進行中 / ⏳保留中
- システムアイコン: 固定（⚙️🛠️💾🔧📦等）
- 例: ✅⚙️ ツール横断フレーム — 3カテゴリフレームワーク確立
- 例: 🚀🛠️ GitHub Actions整備 — 自動テスト・デプロイ

========================================
## 優先度別アクション

1. 種別 1・2 が Notion に登録 → すぐ対応
2. 種別 3 は Chat スレッド相談 → ベストエフォート対応
3. 種別 1 の期限迫っていたら → 種別 2・3 は一時停止

========================================
## UI 表示対策

- 最初の 20 文字で「何をするのか」が一目でわかる
- 「—」で区切る
- 詳細は「—」以降に記載

例:
✅📋 LOM発送準備 — 資料・スケジュール確認
↑    ↑                ↑
進捗  性質            詳細

========================================
## よく使うアイコン

【性質アイコン（種別 1・2）】
- 📋 チェックリスト・確認
- 📊 データ・分析・レポート
- 📝 ドキュメント・資料
- 🔗 リンク・連携
- 🎯 プロジェクト・ゴール

【システムアイコン（種別 3 固定）】
- ⚙️ 設定・構成
- 🛠️ ツール・整備
- 💾 データ・ファイル
- 🔧 メンテナンス
- 📦 パッケージ・モジュール

========================================
## ツール分担

Code: リード・実装・GitHub連携（全種別対応）
Codex: Code のサポート・分析（全種別対応）
Cowork: Notion 更新・ワークフロー管理（種別 1・2 のみ）
