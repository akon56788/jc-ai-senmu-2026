## 📁 プロジェクト概要

あきる野青年会議所・専務理事として、プロジェクト型業務と定常業務が混在する環境での「タスク死＆混乱」から脱却するためのAI活用システム。

Google Drive フォルダ「専務理事対応（AI活用）」を **SSOT（Single Source of Truth）** とし、各AIツールが共通コンテキストファイルを参照して動作する。

---

## 🗂️ SSoT ファイル構成

### 上位設計（全ツール共通）

| ファイル名 | 役割 | 更新頻度 |
|-----------|------|---------|
| README.md | このファイル。プロジェクト全体の入口 | 随時 |
| MANIFEST | 全ファイルの管理台帳（バージョン・Drive更新日・GitHub path） | タスク完了時 |

| VISION.md | プロジェクトビジョン・実現技術・ユースケース | 四半期 |
| DRIVE_SYNC_STATUS.md | Drive↔GitHub同期状況 | 随時 |
| DRIVE_SYNC_CHECKLIST.md | Drive同期チェックリスト | 随時 |
| THREAD_NAMING_GUIDE.txt | スレッド命名ガイド | 稀 |
| notion_poc_design.md | Notion PoC設計 | 随時 |

## 🔖 スレッド命名規則（3カテゴリフレーム）

### カテゴリ構造

```
[進捗アイコン][タイプアイコン] タイトル — サブタイトル
```

### 進捗アイコン

| アイコン | 意味 |
|---------|------|
| ✅ | 完了 |
| 🔵 | 進行中 |
| ⏳ | 保留・待ち |

### タイプアイコン

| アイコン | 意味 |
|---------|------|
| 🎯 | プロジェクト |
| 🔄 | 定常業務 |
| 🛠️ | システム・ツール整備 |
| 📋 | ドキュメント・設計 |
| 💬 | 相談・検討 |

### 例

```
✅🎯 JCカップ — 参加申込フォーム作成
🔵🛠️ SSoT整備 — MANIFEST v1.2 + GitHub同期準備
⏳🔄 LOM発送 — 5月度（Round 4）
```
