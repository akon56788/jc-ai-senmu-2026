## Codex での作業フロー

### 開始時（毎回）
```
1. SHARED_SOURCE.txt を読む
2. CLAUDE_enhanced.md で全体方針確認
3. AGENTS.md.txt で Codex 役割・LOM ルール確認
4. 作業開始
```

### LOM 発送作業時
```
1. 上記「開始時」を実行
2. AGENTS.md.txt の「LOM 発送仕様」セクション読む
3. CODEX_WORKFLOW.md.txt で検証基準確認
4. Excel テンプレート修正・検証
5. ユーザーに成果物を報告
```
